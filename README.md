# traffic-signal-ai
AI-Based Traffic Signal Control System using Reinforcement Learning
🚦 Traffic Signal Control using Reinforcement Learning

📌 Problem

Traditional traffic signals use fixed timers, which cause congestion.

💡 Solution

We built an AI-based traffic signal system using Reinforcement Learning that dynamically adjusts signals based on traffic conditions.

⚙️ How it works

- Environment simulates traffic flow
- Agent controls signal decisions
- Reward is based on minimizing traffic congestion

🧠 Algorithm

Q-Learning

▶️ How to Run

1. Install dependencies:
   pip install -r requirements.txt

2. Train the model:
   python train.py

3. Run the app:
   python app.py

Tech Stack

- Python
- NumPy
- Gradio
- Reinforcement Learning
